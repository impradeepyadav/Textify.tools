Hex Text Converter

Convert text to hexadecimal and hex back to text instantly. All processing happens in your browser – no data is ever uploaded.

How to Use the Hex Text Converter

  1. Select conversion mode: Choose "Text → Hex" to encode text, or "Hex → Text" to decode hex into readable characters.
  2. Adjust formatting options: For text‑to‑hex, pick an output style (spaced, continuous, dash‑separated, etc.). For hex‑to‑text, choose how to interpret the input (auto‑detect or strict).
  3. Set the encoding: Use UTF‑8 (recommended for multilingual text) or ASCII for legacy data.
  4. Check optional boxes: Toggle uppercase hex digits or remove spaces for a compact result.
  5. Click “Convert”: The result appears instantly in the output box.
  6. Copy or download: Save the result as a .txt file or copy it directly to your clipboard.

💡 Pro tip: Use the “Auto‑detect” prefix mode when converting hex strings that may include “0x”, spaces, or dashes. Our tool automatically normalises the input before decoding, giving you more flexibility.

  • 100% private: All conversions happen in your browser. Your data never leaves your device.

  • Supports multiple formats: Handles continuous hex, spaced, dashed, colon‑separated, and 0x‑prefixed strings.
  • No installation: Works directly in any modern browser on desktop or mobile.

Frequently Asked Questions

Is this converter really free?
Yes, completely free with no limits. No account, no registration, and no hidden fees.
What does “Auto‑detect” do for hex input?
It automatically strips common hex prefixes (like 0x), spaces, dashes, and colons, then decodes the remaining hex pairs. This means you can paste any typical hex representation without manual cleanup.
Does it support non‑ASCII characters?
Yes. When you set the encoding to UTF‑8, the tool correctly handles all Unicode characters (emojis, accented letters, etc.).
Is my text private?
Absolutely. All processing is client‑side. Your text is never uploaded to any server.
What if my hex input is invalid?
The tool will show an alert. Make sure your hex string contains an even number of valid hex characters (0‑9, A‑F). For decoding, avoid mixing non‑hex characters unless you use the “Strip all” option.
