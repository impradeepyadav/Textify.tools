Morse Code Converter

Convert text to Morse code and decode Morse back to plain text. All processing happens in your browser – private, fast, and simple.

How to Use the Morse Code Converter

  1. Enter your text or Morse code: Paste or type the string you want to convert.
  2. Choose the direction: Select Text to Morse to encode, or Morse to Text to decode.
  3. Click "Convert": The result appears immediately in the output area.
  4. Copy or download: Use the copy button to grab the result or save it as a .txt file.
