Unicode Converter

Convert text to Unicode code points (decimal, hex, binary, HTML) and decode Unicode back to text. All processing happens locally in your browser.

What is Unicode?

Unicode is a universal character encoding standard that assigns a unique number (code point) to every character, symbol, and emoji across all writing systems. It allows computers to represent and manipulate text consistently. For example:

  • 'A' → U+0041 (hex), 65 (decimal)
  • '★' → U+2605 (hex), 9733 (decimal)
  • '😀' → U+1F600 (hex), 128512 (decimal)

Text → Unicode (encode): Converts each character to its numeric representation (decimal, hex, binary, or HTML entity). Useful for debugging, web development, and data processing.

Unicode → Text (decode): Converts code point sequences back to readable text. Accepts formats like 65 66 67, U+0041 U+0042, &#65;, or 0b1000001.

⚡ Client-side only: All conversions run in your browser – your data never leaves your device.

How to Use the Unicode Converter

  • Choose a conversion direction: "Text → Unicode" to encode, or "Unicode → Text" to decode.
  • Select an output format for encoding: decimal, hex (with U+ prefix), binary, or HTML entities.
  • Paste your input (plain text or Unicode code points) into the text box.
  • Click "Convert" – the result appears instantly in the output area.
  • Copy or download the result for further use.

Pro tip: If you need to decode, make sure you use a supported format (e.g., "65 66 67" for decimal, "U+0041 U+0042" for hex with U+ prefix, or "0b1000001" for binary). Spaces or commas are allowed as separators.

Frequently Asked Questions

What Unicode code point formats are supported for decoding?
The decoder understands decimal numbers (e.g., 65 66), hexadecimal prefixed with U+ (e.g., U+0041), binary prefixed with 0b (e.g., 0b1000001), and HTML decimal entities (e.g., &#65;). Separators like spaces, commas, or newlines are allowed.
Does this tool support emojis and non-Latin scripts?
Yes, full Unicode support includes emojis (😀, 🚀), CJK ideograms, accented letters, and symbols from all languages. Every character has a unique code point.
Is my data private?
Absolutely. All processing happens locally in your browser using JavaScript. No server uploads, no logs, no tracking – your text is never sent anywhere.
What are HTML entities used for?
HTML entities are often used to embed Unicode characters in web pages where direct typing may be ambiguous. For example, you can write &#128512; to display 😀.
