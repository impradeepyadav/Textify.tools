Frequently Asked Questions

Is this SQL formatter free?
Yes, completely free with no limits. No registration, no premium tiers.
Does it work offline?
Yes, after the page loads, the formatter works entirely offline – no internet connection required.
What SQL dialects are supported?
We support Standard SQL, MySQL, PostgreSQL, SQL Server (T-SQL), Oracle PL/SQL, MariaDB, SQLite, BigQuery, Snowflake, Redshift, Spark SQL, Hive, DB2, N1QL, Trino, TiDB, and SingleStoreDB.
Is my SQL query private?
Absolutely. All processing happens in your browser. No data is ever sent to our servers.
What is placeholder replacement?
You can replace `?` or `:param` placeholders with actual values. Enter comma-separated values in the placeholder field, and they'll be applied in order during formatting.
Can I minify my SQL instead of beautifying?
Yes, switch the Operation Mode to "Minify / Compact" to remove unnecessary whitespace and create a compact SQL string.
Does it support stored procedures?
The formatter works well with most SQL constructs, but may not fully support complex stored procedure syntax. For regular SELECT, INSERT, UPDATE, DELETE statements it works perfectly.
